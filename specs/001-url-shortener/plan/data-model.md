# Data Model: URL Shortener Service

## Entities

### Link

Primary entity for URL shortening.

| Field | Type | Constraints | Notes |
|-------|------|-------------|-------|
| Id | GUID | Primary Key, required | Auto-generated by Entity |
| Sequence | int | Alternate Key, required | Auto-generated by database, required for key pagination |
| OriginalUrl | string | required, validated | Must be valid URL format |
| ShortCode | string | unique, indexed | Case-insensitive storage |
| CreatedAt | DateTimeOffset | required | Auto-set by Entity, default: DateTimeOffset.UtcNow |
| ExpiresAt | DateTimeOffset? | optional | Auto-set by Entity, default: CreatedAt + 30 days |
| IsActive | boolean | required, default: true | Soft delete flag |
| QrCodeUrl | string? | optional | URL to QR code image |

### Relationships

- Link has no foreign keys (simplified MVP)
- No User entity (anonymous)

## Validation Rules

### Link.OriginalUrl
- MUST be a valid URL format (URI)
- MUST be absolute (have scheme)
- MAX length: 2048 characters

### Link.ShortCode
- MUST be alphanumeric only
- MUST be 8 characters (default)
- MUST be unique
- MUST be case-insensitive (store uppercase, compare uppercase)
- MAX length: 16 characters

### Link.ExpiresAt
- MUST be greater than CreatedAt (if set)
- Default: CreatedAt.AddDays(30)
- Future: Authenticated users can set custom

## State Transitions

| State | Valid Transitions |
|-------|-----------------|
| Created | Active → Expired (automatic) |
| Active | → Inactive (manual soft-delete) |
| Expired | Terminal state (shows error) |

## Indexes

| Index | Fields | Type | Purpose |
|-------|--------|------|---------|
| UQ_Links_ShortCode | ShortCode | Unique | Fast redirect lookup |
| IX_Links_ExpiresAt | ExpiresAt | Non-unique | Cleanup queries |
| IX_Links_IsActive | IsActive | Non-unique | Active link filter |

## Database Configuration

- Provider: SQL Server
- ORM: Entity Framework Core
- Migrations: Code-first via EF Core

## Notes

- No analytics tables (out of scope)
- No user tables (anonymous)
- Simple single-table design for MVP
